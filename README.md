# Netz OÖ Smart Meter

Home Assistant HACS Integration für das Netz Oberösterreich Smart Meter Portal.

## Funktionen

- Login ins Netz OÖ eService
- 15-Minuten-Verbrauch
- Tages-, Monats- und Jahresverbrauch
- Zählerstand
- Energy Dashboard
- Mehrere Zählpunkte (geplant)

## Installation

1. Repository über HACS hinzufügen.
2. Integration installieren.
3. Home Assistant neu starten.
4. Benutzername und Passwort eingeben.
